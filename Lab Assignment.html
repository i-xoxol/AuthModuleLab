Lab Assignment: User Authentication

This lab is designed to help you understand the principles of user authentication in a software application. You will be working with a simple Python-based command-line application that demonstrates user registration and login functionalities.

Objectives

  • Understand the basic concepts of user authentication.
  • Learn how to implement user registration and login features.
  • Understand the importance of secure password handling (password hashing).
  • Gain experience with reading from and writing to a simple file-based database (JSON).

Instructions

You are provided with a set of Python files that make up the authentication lab. Your task is to complete the missing code in the `auth.py` file.

  1. Review the code: Start by reviewing the provided files:
    • main.py: The main entry point of the application.
    • user.py: The User class representing a user.
    • user_registration.py: Handles user registration.
    • auth.py: This is the file you will be modifying. It contains the core authentication logic.
    • test_auth.py: Contains unit tests to verify your implementation.
    • usersdb.json: A simple JSON file to store user data.
  2. Implement the `login` function in `auth.py`: Open the `auth.py` file and implement the `login` function. This function should take a username and password as input, and it should return `True` if the credentials are valid, and `False` otherwise. You will need to:
    • Read the user data from the `usersdb.json` file.
    • Find the user with the given username.
    • Compare the provided password with the stored hashed password.
  3. Run the tests: Once you have implemented the `login` function, run the unit tests in `test_auth.py` to verify your implementation.
  4. Run the application: Run the `main.py` file to test the application manually.

Submission

Once you have completed the lab, submit the `auth.py` file.
